O que é o projeto?

Somos integrantes de um curso técnico do Senai (Serviço Nacional de Aprendizagem Industrial) chamado "Desenvolvimento de Sistemas", e nos foi proposto o desafio de criar um site com dados referentes à pandemia de Sars-Cov-2, causador da doença Covid-19. Para tal feito, estudamos, entre aulas teóricas e práticas, a implementação de dados API "Application Programming Interface" (Interface de Programação de Aplicação), que nada mais é que a comunicação entre sistemas. As API's permitem a integração entre dois, ou mais, sistemas, onde um deles fornece dados para o receptor, a fim do receptor realizar funções específicas com os dados do fornecedor. O mais importante é a COMUNICAÇÃO EM TEMPO REAL, ou seja, se atualizarem os dados do fornecedor, os dados do receptor são atualizados automaticamente.

Quais seriam os objetivos?

Além do aprendizado acerca das API's, ficam mais dois objetivos principais: informar e disponibilizar. No quesito de informar, a apresentação dos dados é o principal ponto central, conectando os servidores e recebendo os dados. Já na parte de disponibilização, fomos instruidos de como deixarmos nosso site público para que todas as pessoas que possuam acesso à internet possam ver nosso trabalho. Se você pode ler meu site, agradeça aos nossos professores que nos ensinaram a publicar este site em um servidor.

Quais foram as tecnologias utilizadas?

O atual assunto de nossas aulas no Senai é programação em JavaScript (que é diferente de Java). Por meio dele, podemos criar maiores interações no site, assim como podemos armazenar e manipular dados. Uma dessas formas é utilizando os API's. Também utilizamos nossos conhecimentos prévios em programação de HTML e CSS. O HTML basicamente cria o "esqueleto" do site e o CSS dá a aparência, cria a "pele" desse "esqueleto". Também utilizamos um servidor de hospedagem chamado "000webhost", um servidor de hospedagem gratuito e ideal para as intenções do site. Por fim, utilizamos outro "servidor" chamado "Git Hub", um site que hospeda backups das programações e consegue compartilhar elas com usuários do site, desde que cadastrados. É possível o trabalho em equipes no "Git Hub", porém, este site foi desenvolvido sozinho.
